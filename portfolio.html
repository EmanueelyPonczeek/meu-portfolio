    Website para panificadora Dolce Aroma

    Muitas vezes, quando quero tomar um café em um lugar diferente, me vejo correndo atrás de informações como localização, imagens dos produtos, horário de funcionamento, preços e cardápio, ou até mesmo tendo que ligar para o estabelecimento para obter essas informações. Com base nisso, decidi criar um website para uma padaria fictícia, incorporando elementos que considero importantes para despertar o interesse do usuário e facilitar sua busca. Para este projeto, apliquei conceitos de UX que desenvolvi no curso da EBAC, como a criação de uma protopersona do estabelecimento, estudo de cores, tipografia, benchmarking e definição do tom de voz. Além disso, utilizei elementos de UI para garantir que o site fosse de fácil navegação e intuitivo.

    Projeto de Estudo JBL 520 Tune

    Este foi o primeiro website que desenvolvi sozinha, e para o projeto, escolhi um produto que utilizo bastante: o JBL 520 Tune, um headphone Bluetooth. O objetivo principal deste site foi criar uma página informativa, onde os usuários podem acessar todas as especificações e detalhes sobre o produto. Além disso, o site foi projetado com elementos estratégicos que conduzem o usuário de forma eficiente à página oficial de compra da JBL, proporcionando uma navegação fluida e intuitiva.
